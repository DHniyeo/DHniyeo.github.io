# 개요


각 자료구조의 특징과 예시를 알아보기 위해 한번 정리해 보았다.


## **1. 배열 (Array)**

- **특징:**
	- 고정된 길이의 요소를 저장
	- 인덱스를 이용하여 빠른 랜덤 접근 가능 (O(1))
	- 순차적 접근 불편
	- 추가/삭제 비효율적 (O(n))
- **장점:**
	- 랜덤 접근 속도 빠름
	- 메모리 할당 방식 간단
- **단점:**
	- 길이 변경 불가능
	- 추가/삭제 비효율적
	- 메모리 낭비 가능성 있음

> 💡 브루트포스 알고리즘


> **Memory   1112KB                                   Time   8ms                                Code Length   318B**



{% raw %}
```c++
#include<stdio.h>

int main()
{
	int n;
	int result = 0;
	scanf("%d", &n);


	for (int i = 0; i <= n; i++)
	{
		int sum = 0;
		int tmp = i;
		sum += tmp;

		// 245
		while (tmp / 10) {
			sum += tmp % 10;
			tmp = tmp / 10;
		}
		sum += tmp;

		if (sum == n) {
			result = i;
			break;
		}
	}
	printf("%d", result);

}
```
{% endraw %}



사용자로부터 정수를 입력받아 해당 정수가 입력된 수와 각 자리수의 합이 같은 수를 찾는 프로그램이다. 입력된 수부터 시작하여 각 자리수의 합을 구하고, 입력된 수와 비교하여 같은 경우 해당 수를 결과로 출력한다.

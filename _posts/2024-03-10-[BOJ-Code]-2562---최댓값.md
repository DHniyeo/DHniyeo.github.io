```c++
#include <stdio.h>

int main()
{
	int max[9];

	for (int i = 0; i < 9; i++) {
		scanf(" %d", &max[i]);
	}

	int max_v = 0;
	int max_i = 0;
	for (int i = 0; i < 9; i++)
	{
		if (max_v < max[i]) {
			max_v = max[i];
			max_i = i;
		}
	}
	printf("%d\n", max_v);
	printf("%d\n", max_i+1);

}
```
{% endraw %}



이 코드는 9개의 숫자를 입력받아서 그 중에서 가장 큰 숫자와 해당 숫자의 인덱스를 출력하는 프로그램이다. 먼저 9개의 숫자를 입력받은 후, 입력된 숫자들 중에서 가장 큰 숫자와 그 숫자가 몇 번째로 입력되었는지를 찾아서 출력한다.

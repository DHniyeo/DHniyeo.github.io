## 개요


Github에서 파일 받는 방법은 터미널에서 git clone을 통해 통째로 파일을 받아오거나, curl 또는 wget을 통해 파일을 하나 씩 다운 받는 방법이 있다.


물론 둘 다 편한 방법이지만 생각이 안 날 때도 있고 여러모로 번거로울 때가 있다.


그래서 편하게 사용 가능한 사이트가 있어 소개하고자 한다.
